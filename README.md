# AED 模擬教學器

本專案為 AED 數位模擬教學系統，用於急救教育、AED 操作流程演練與 CPR 節拍訓練。

## 功能特色

- 支援手機與電腦瀏覽器
- 可離線使用
- 支援 PWA 安裝到手機桌面
- AED 操作流程：開 → 貼 → 插 → 電
- 支援需要電擊 / 不需要電擊情境
- CPR 2 分鐘節拍提示
- 教官可長按標題 3 秒進入語音修正面板
- 建議使用 Safari / Chrome，不建議使用 LINE 內建瀏覽器

## 使用方式

1. 開啟 index.html 或 GitHub Pages 網址
2. 點選「進入模擬教學並啟用聲音」
3. 依序操作 AED 流程
4. 如需手機安裝，可使用瀏覽器「加入主畫面」

## 注意事項

本系統僅供教育訓練與模擬教學使用，不能取代真實 AED 設備、正式急救訓練或醫療判斷。
